## Keybinds/HotKeys
`L`: Rename
`Ctrl+L`: Retype
`G`: Goto
`;`(semicolon): Comment
### Recommended Settings
`Edit` -> `Tool Options` -> `Listing Fields` -> `Cursor Text Highlight` -> `Mouse Button to Activate = LEFT`
